## Primary data 

Data sources can be divided into primary and secondary sources. 

Primary data is the type of data, which is gathered for specific purpose. For instance, Wal-Mart is using a face-recognition app, which identifies the expressions of customers in real-time and messages to employees in a case of insatisfied customer - as the insatisfied customer is more likely to need help to fulfill ones needs.

This post is mainly focusing on the secondary data sources. If you wanna see how primary data is collected to from the idea to the practice, check this blog post. By using SMART-principle, it illustrates how business operating in food industry can track whether it's product fulfills the predefined requirements.
